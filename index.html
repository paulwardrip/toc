Paul Wardrip's Git Projects

Newer Libraries (npm)

  • bookmarklet-compiler [source] (npm: bookmarklet-compiler) compiles a node.js script into a bookmarklet ready to be pasted into a bookmark in your browser
  • hyperfocus-armyknife [source] (npm: @hyperfocus/armyknife) A collection of small utility classes to be imported as a whole, or individually as the user sees fit.
  • xmlschema.js: [source] (npm: xmlschema.js) XML schema (xsd) validation written in pure Javascript, usable in the browser, node.js / ES6.

Older Libraries (bower)

  • auto-lightbox: [source] (bower: auto-lightbox) a lightbox that automatically shows and hides when the element set as its parent shows or hides, can be assigned to different parent elements on the fly.
  • dots!: [source] (bower: dots) an fully CSS based spinner for jQuery that displays three animated dots.
  • centerize: [source] (bower: centerize) a jQuery plugin for sizing or positioning elements on the page based on percentages of the height or width of a parent (or other specified) element.
  • fieldvalidator: [source] (bower: fieldvalidator) A super-slick realtime validation system for jQuery, supporting required fields with the required attribute and regex patterns with the pattern attribute. Validates as the user stops typing or moves to the next field. Valid or invalid status support messages and are shown with animated color changes based on status. Colors and delays are configurable.

Applications

  • GoogleExportSMS: [source] allows you to convert your Google Voice/Hangouts messages into SMS messages that can be imported into your Android phone.
  • music-video-jukebox: [source] Warning: Youtube APIs have changed and broken this application, I haven't gotten around to updating it.
    add videos from youtube and this app will play you random songs from your library, you can skip songs. You can pick a song you want to hear, by adding to the queue or you have the option to play immediately. You can download a backup of your library and import it later, since the library is kept in localStorage. This app is very, very functional ... but maybe not quite polished and complete. You may have to reload the browser to get it playing after you add your first set of songs. Enjoy!
