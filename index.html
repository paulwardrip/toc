Paul Wardrip's Git Projects

Applications

  • GoogleExportSMS: [source] allows you to convert your Google Voice/Hangouts messages into SMS messages that can be imported into your Android phone.

Libraries

  • xmlschema.js: [source] (bower: xmlschema.js) xml schema (xsd) validation written in pure Javascript, usable in the browser. Nearly all xsd features that are necessary for the majority of use cases have been implemented.
  • auto-lightbox: [source] (bower: auto-lightbox) a lightbox that automatically shows and hides when the element set as its parent shows or hides, can be assigned to different parent elements on the fly.
  • dots!: [source] (bower: dots) an fully CSS based spinner for jQuery that displays three animated dots.
  • centerize: [source] (bower: centerize) a jQuery plugin for sizing or positioning elements on the page based on percentages of the height or width of a parent (or other specified) element.
  • journal: [source] (bower: journal) a logging framework for the console log allowing you to turn on and off log messages at the levels: debug, info, warn, error. Logs for all users start at warn and are only turned to info or debug for a user that turns those levels on. Allows you to set the level for a specific js file or function name. Using this framework you can leave in verbose debugging information without it being displayed in the console for your users.
  • fieldvalidator: [source] (bower: fieldvalidator) A super-slick realtime validation system for jQuery, supporting required fields with the required attribute and regex patterns with the pattern attribute. Validates as the user stops typing or moves to the next field. Valid or invalid status support messages and are shown with animated color changes based on status. Colors and delays are configurable.
